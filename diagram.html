Class Diagram for Bowling App

Each Frame has one (when started) or two shots, except the tenth, which can have a third.

classDiagram PlayerGame "1" --> "0..*" GameStats Player "1" --> "0..*" PlayerGame GameStats "1" --> "0..*" Frames class Player{ name: String } class Frames{ shots: List[List[Int]] scores: List[Int] } class GameStats{ player: String frames: Frames } class PlayerGame{ name: String shots: List[List[Int]] frames: Frames gameStats: GameStats }
